Kitura-Sampler
===

Code examples of Kitura

## Usage

```
$ git clone https://github.com/akimacho/Kitura-Sampler.git
$ cd Kitura-Sampler/Basic
$ ./run.sh
```

Then, Open your browser at http://localhost:8090

## Contents

* Basic
* Routing
* Controller
* Template
* Redis
